# 网站使用指南

- 网站地址：https://personal-blog-pi-jade.vercel.app
- 代码仓库：https://github.com/maxwellzfl/personal-blog
- 部署平台：https://vercel.com（免费 Hobby 计划）

工作原理一句话：文章是仓库里的 Markdown 文件，每次向仓库提交改动，Vercel 会自动重新构建并发布网站，约 1 分钟生效。

## 一、写新文章（最常用）

1. 打开仓库，进入 `src/content/posts/` 目录
2. 点右上角 **Add file → Create new file**
3. 文件名用英文，如 `my-first-post.md`（文件名会成为文章网址）
4. 按下面模板写内容
5. 点右上角 **Commit changes...** → 再点弹窗里的 **Commit changes**

文章模板：

```md
---
title: 文章标题
date: 2026-07-10
category: 投资
description: 一句话摘要（可选）
draft: false
---

正文用 Markdown 写。支持标题、引用、表格、代码块、图片。
```

字段说明：

| 字段 | 说明 |
| ---- | ---- |
| title | 文章标题，必填 |
| date | 日期，格式 `2026-07-10`，必填 |
| category | 只能填【投资】【生活】【AI】之一，必填 |
| description | 摘要，显示在 RSS 里，可选 |
| draft | 填 `true` 则不发布（存草稿），默认 `false` |

## 二、修改或删除文章

打开文章文件 → 点右上角铅笔图标编辑，或点 ··· 选 Delete file → 提交。

## 三、文章里插图

1. 进入 `public/` 目录 → **Add file → Upload files**，上传图片（如 `photo.jpg`）
2. 文章里写：`![图片说明](/photo.jpg)`

## 四、修改站点信息

| 要改什么 | 改哪个文件 |
| ---- | ---- |
| 站点名称、简介 | `src/consts.ts` |
| 关于页自我介绍 | `src/pages/about.astro` |
| 颜色、字体、版式 | `src/styles/global.css` |
| 域名（影响 RSS 链接） | `astro.config.mjs` 里的 `site` |

## 五、绑定自己的域名（可选）

1. 在 Vercel 打开 personal-blog 项目 → **Settings → Domains** → 添加域名，按提示到域名商配置 DNS
2. 绑定后把 `astro.config.mjs` 里的 `site` 改成新域名

## 六、本地预览（进阶，可选）

电脑装 Node.js 20+（https://nodejs.org），然后：

```bash
git clone https://github.com/maxwellzfl/personal-blog.git
cd personal-blog
npm install
npm run dev
```

浏览器打开 http://localhost:4321。改完用 git push 推送，网站自动更新。

## 七、常见问题

**提交了但网站没更新？** 到 vercel.com 打开项目的 Deployments 页面看构建状态；失败通常是文章开头的 frontmatter 格式错了（category 拼写、日期格式、缺少 `---`）。

**文章不显示？** 检查是否 `draft: true`，以及 category 是否为 投资/生活/AI 之一。

**RSS 订阅地址？** https://personal-blog-pi-jade.vercel.app/rss.xml
